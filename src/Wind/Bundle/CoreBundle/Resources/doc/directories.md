# Wind Directory Structure

	- app (system stuff)
		- cache
		- config
		- logs
		- AppKernel.php
		- autoload.php
		- bootstrap.php.cache
		- console
	- src
		- SiteName
			- Bundle
				- ... custom site bundles ...
				- SiteThemeBundle
					- Wind.yml (contains information about the bundle)
					- Resources
						- public
							- css
							- js
							- img
						- views (if defined as theme, these views override any other ones)
							-
		- Wind (system stuff)
			- Bundle
				- CoreBundle ...
	- vendor ... (system stuff, and third-party bundles)
	- web ... (system stuff)
